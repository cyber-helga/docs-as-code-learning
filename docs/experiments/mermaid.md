# Mermaid diagrams

Simple flowchart

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

Timeline diagram

```mermaid
timeline
    title History of social media platforms
    2002 : LinkedIn
    2004 : Facebook
         : Google
    2005 : YouTube
    2006 : Twitter
    2018 : TikTok
```

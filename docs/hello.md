# Introduction

Hello and welcome.

My name is Olha (also known as *Cyber Helga*). I’ve been working as a technical writer for nearly six years, and I’m still driven by the same question: how do I keep getting better at this craft?

Lately, that question has become more interesting with the rise of AI.

This website is my playground. It’s where I explore how modern technical writing comes together—from setting up a documentation site with Docusaurus, to hosting it on GitHub Pages, to automating builds and deployments with GitHub Actions. The process behind it was a collaboration between me and ChatGPT, which guided me through everything from local setup to production.

But this site isn’t about tools.

It’s about writing.

Technical writing is evolving. We now rely on tools to proofread, restructure, analyze, and scale our work. But the core responsibility hasn’t changed. We are still:

- the source of clarity  
- the authors of the first draft  
- the bridge between technical teams and users  
- the ones who ask questions others don’t  

Tools can support the process, but they don’t replace judgment, context, or intent.

This website is my attempt to push further.

- Docs-as-code → explored  
- AI-assisted writing → applied  
- Diagrams and visual thinking → in progress  
- Documentation audits → tested  
- Content creation → ongoing  

So what’s next?

That’s exactly what I’m here to find out.
